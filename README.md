# Magento Test Automation Framework

- This project is an automated testing framework for the Magento website, built with:
  - Java 21
  - Selenium WebDriver
  - TestNG
  - Maven
  - Page Object Model (POM)
  - Allure Report
## How to Run
### i. Install Requirements
- Java 21  
- Maven 3+  
- Chrome / Firefox / Edge  
### ii. Run Tests
- mvn test verify
### iii. Generate Allure Report
- Create the report: mvn allure:report
- Open the report: mvn allure:serve
